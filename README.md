## Getting Started

run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


run storybook : 
```bash
npm run storybook
```

Open [http://localhost:6006](http://localhost:6006) with your browser to see the result.


Run local API:
```bash
npm run mock-api
```
